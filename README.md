# NG前端脚手架0.1.0版本 #

## 框架和工具库 ##
*  angular 1
*  underscore
*  jquery

## 自动化工具 ##
* npm
* bower
* grunt

>* 默认启动开发模式
>* produce 生成生产环境版本
>* viewproduce 调试生产环境版本
>* genDoc 生成[jsdoc](http://usejsdoc.org/)



## 命名规则(基于angular) ##
> 以下用test为例
>> 基础js(./script)

>> service(./service)
>>> 命名为testService.js
>>>服务名为testService

>> controller(./controller)
>>> 命名为testCtrl.js
>>> 控制器名为testCtrl.js

>> directive(./directive)
>>> html命名为./directive/html/test.html
>>> js命名为./directive/js/test.js
>>> less命名为./directive/less/test.less
